# Workflow: /feature
1. **Research:** Analyze existing components for design patterns.
2. **Plan:** Generate an `Implementation Plan` artifact.
3. **Build:** Execute the code changes in the Editor View.
4. **Lint:** Run `npm run lint` and auto-fix errors.
5. **Verify:** Launch the **Browser Subagent** to record a `Walkthrough` of the UI.
